### How to use Devtools in chrome extension

1.일렉트론 버젼의 devtools를 설치

```
yarn global add @vue/devtools
```

2. 빌드된 상태의 크롬익스텐션 아르고커넥트 실행

3. devtools 실행

```
vue-devtools
```
